## Installing on Local Machine

* Fork to get your own copy of the project 
* Clone the repo
* `cd VideoVerse/`
* `npm install`
* `npm start/nodemon`

#### Build and load the app
* `npm install`
* `npm start`
* run `localhost:3000` in the browser
